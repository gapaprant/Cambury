# cross_spec_glossary

| Termo canônico | Definição |
|---|---|
| workspace | Unidade organizacional de conversa e workflow |
| conversation | Sessão de interação com trilha persistida |
| workflow | Execução estruturada com estados e tasks |
| task | Unidade mínima de trabalho dentro de workflow |
| session_ledger | Resumo persistido por sessão/conversa |
| baseline_v1 | Linha base formal para implementação |
